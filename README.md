# System Mono

A minimal Chrome extension to force monospace fonts on any webpage.

## How to Install
1. Open `chrome://extensions/` in Chrome.
2. Enable **Developer mode** in the top right.
3. Click **Load unpacked** and select this project folder.

## Key Controls
- **Global Toggle**: Apply monospace to all websites.
- **Current Tab**: Override settings for the active page.
